# Leaflet Map Plugin for WordPress

Custom WordPress plugin that adds an interactive map using the Leaflet library.  
Insert a map anywhere with a simple shortcode: `[leaflet_map]`.

## Features

- Lightweight, no external API keys required (uses free Leaflet tiles)
- Responsive map that adapts to any container width
- Customizable marker with popup text
- Easy to extend (add more markers, change tile layers, etc.)

## Installation

1. Upload the `leaflet-map-plugin` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode `[leaflet_map]` in any post, page, or widget

## Customization

You can modify the default coordinates and popup text directly in the plugin code, or extend it via hooks (coming soon).

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
